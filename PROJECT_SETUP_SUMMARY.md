**שאלה:** האם אפשר להשתמש באותו API לכמה פרויקטים?

**תשובה:** ✅ כן!
- אפשר להשתמש באותו WhatsApp Business Account לכמה פרויקטים
- כל פרויקט יכול להיות Phone Number ID נפרד (אם יש כמה מספרים)
- או אותו Phone Number ID עם webhooks שונים (מומלץ: `/webhook/solanahunter`)
- חלופות: מספרים נפרדים או App נפרד ב-Meta Business

**החלטה:** נשתמש באותו Phone Number עם webhook endpoint ייעודי
