# 🔑 איך לייצא Private Key מ-Phantom (מדריך מפורט)

**אם כבר יש לך ארנק ב-Phantom ורוצה לייצא את ה-Private Key**

---

## 📱 שלב 1: פתח את Phantom

1. פתח את אפליקציית Phantom בטלפון
2. ודא שאתה בארנק הנכון (הארנק שיצרת לבוט)

---

## ⚙️ שלב 2: לך להגדרות

### דרך 1 (הכי נפוץ):
1. לחץ על **⚙️ Settings** (האייקון התחתון)
2. בחר **"Security & Privacy"** או **"אבטחה ופרטיות"**

### דרך 2 (אם לא מוצא):
1. לחץ על **שם הארנק** (בחלק העליון)
2. בחר **"Advanced"** או **"מתקדם"**
3. בחר **"Export Private Key"** או **"ייצא מפתח פרטי"**

### דרך 3 (אם עדיין לא מוצא):
1. לחץ על **⚙️ Settings**
2. גלול למטה
3. חפש **"Show Private Key"** או **"הצג מפתח פרטי"**

---

## 🔐 שלב 3: ייצוא Private Key

1. **תבקש ממך להזין סיסמה:**
   - הזן את הסיסמה של הארנק (6 ספרות או יותר)
   - לחץ **"Confirm"** או **"אישור"**

2. **Phantom יציג את ה-Private Key:**
   - זה מחרוזת ארוכה של תווים
   - למשל: `5KJvsngHeMoi...` (ארוך מאוד, 88 תווים בערך)
   - **⚠️ זה רגיש מאוד - אל תשתף עם אף אחד!**

3. **העתק את ה-Private Key:**
   - לחץ על **"Copy"** או **"העתק"**
   - או העתק ידנית (הכל!)

---

## ⚠️ חשוב לדעת:

### מה זה Private Key?
- זה המפתח שמאפשר לחתום על טרנזקציות
- זה מה שהבוט צריך כדי לקנות/למכור
- **אם מישהו ישיג אותו → הוא יכול לגנוב את הכסף!**

### מה זה Secret Recovery Phrase (12 מילים)?
- זה המפתח לשחזור הארנק
- אם תאבד את ה-Private Key, זה הדרך היחידה לשחזר
- **זה לא אותו דבר כמו Private Key!**

---

## 📝 שלב 4: הוספה ל-.env

1. **פתח את הקובץ** `backend/.env`
2. **מצא את השורה:**
   ```
   WALLET_PRIVATE_KEY=your_private_key_base58_here
   ```
3. **החלף את** `your_private_key_base58_here` **ב-Private Key שהעתקת:**
   ```
   WALLET_PRIVATE_KEY=5KJvsngHeMoi...המפתח המלא שלך...
   ```
4. **ודא:**
   - אין רווחים לפני או אחרי ה-`=`
   - המפתח המלא (ללא שורות נוספות)
   - שמור את הקובץ (Ctrl+S)

---

## ✅ שלב 5: בדיקה

1. **הרץ את verify_setup.py:**
   ```bash
   cd backend
   python verify_setup.py
   ```
2. **אמור לראות:**
   ```
   Wallet: [OK] Connected! Address: xxxxx... Balance: X.XXXX SOL
   ```

---

## 🆘 בעיות נפוצות

### "לא מוצא Export Private Key"?
**פתרונות:**
1. ודא שאתה בארנק הנכון (החדש שיצרת)
2. נסה דרך 2 או 3 למעלה
3. בדוק את גרסת Phantom (עדכן אם צריך)
4. חלק מהגרסאות - לחץ על שם הארנק → Advanced → Export

### "Private Key לא עובד"?
**פתרונות:**
1. ודא שהעתקת את כל המחרוזת (ללא רווחים)
2. ודא שאין שורות נוספות ב-.env
3. ודא שאין רווחים לפני/אחרי ה-`=`
4. נסה להעתיק שוב

### "איך יודע שזה עובד"?
**פתרונות:**
1. הרץ `python verify_setup.py`
2. אמור לראות balance
3. אם יש שגיאה - בדוק את ה-`.env`

## 📸 איפה זה ב-Phantom? (תמונות)

**אם אתה לא מוצא, נסה:**

1. **פתח את Phantom**
2. **לחץ על ⚙️ Settings** (האייקון התחתון)
3. **חפש:**
   - "Security" או "אבטחה"
   - "Privacy" או "פרטיות"
   - "Export" או "ייצא"
   - "Private Key" או "מפתח פרטי"
   - "Show Private Key" או "הצג מפתח פרטי"

4. **אם עדיין לא מוצא:**
   - לחץ על **שם הארנק** (בחלק העליון)
   - חפש **"Advanced"** או **"מתקדם"**
   - שם אמור להיות **"Export Private Key"**
