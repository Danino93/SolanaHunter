# 🔐 מדריך יצירת ארנק Phantom ייעודי לבוט

**מדריך קצר ליצירת ארנק חדש ב-Phantom וייצוא Private Key**

---

## 📱 שלב 1: התקנת Phantom (אם אין)

1. הורד את Phantom לטלפון:
   - **Android:** [Google Play Store](https://play.google.com/store/apps/details?id=app.phantom)
   - **iOS:** [App Store](https://apps.apple.com/app/phantom-crypto-wallet/id1598432978)

2. פתח את האפליקציה

---

## 🆕 שלב 2: יצירת ארנק חדש

1. **פתח את Phantom**
2. אם יש לך כבר ארנק:
   - לחץ על **"+"** או **"Add Wallet"** או **"Create New Wallet"**
   - בחר **"Create New Wallet"** (לא Import!)
3. אם אין לך ארנק:
   - בחר **"Create New Wallet"**

---

## 🔑 שלב 3: שמירת Secret Phrase (חשוב מאוד!)

1. Phantom יציג לך **12 מילים** (Secret Recovery Phrase)
2. **⚠️ שמור את המילים האלה במקום בטוח!**
   - כתוב אותן על נייר
   - או שמור ב-password manager
   - **אל תצלם! אל תשלח במייל!**
3. לחץ **"I've saved it"** או **"Continue"**
4. תצטרך להזין את המילים שוב כדי לאשר ששמרת אותן

---

## 🔒 שלב 4: יצירת סיסמה

1. בחר סיסמה חזקה (6 ספרות או יותר)
2. הזן את הסיסמה שוב לאימות
3. **⚠️ שמור את הסיסמה במקום בטוח!**

---

## 💰 שלב 5: מימון הארנק

1. **קבל את כתובת הארנק:**
   - פתח את Phantom
   - לחץ על **"Receive"** או **"קבל"**
   - העתק את הכתובת (או סרוק QR)

2. **שלח SOL מהארנק הראשי שלך:**
   - פתח את הארנק הראשי שלך ב-Phantom
   - לחץ **"Send"** או **"שלח"**
   - הדבק את הכתובת של הארנק החדש
   - שלח **0.5-1 SOL** (לבדיקות וגז)
   - לחץ **"Send"** ואישור

3. **חכה שהטרנזקציה תאושר** (כמה שניות)

---

## 📤 שלב 6: ייצוא Private Key

1. **פתח את הארנק החדש** ב-Phantom
2. לחץ על **⚙️ Settings** (הגדרות) - האייקון התחתון
3. בחר **"Security & Privacy"** או **"אבטחה ופרטיות"**
4. בחר **"Export Private Key"** או **"ייצא מפתח פרטי"**
5. הזן את הסיסמה של הארנק
6. **העתק את ה-Private Key** - זה מחרוזת ארוכה של תווים (Base58)
   - למשל: `5KJvsngHeMoi...` (ארוך מאוד!)
   - **⚠️ זה רגיש מאוד - אל תשתף עם אף אחד!**

---

## ⚙️ שלב 7: הוספה ל-.env

1. **פתח את הקובץ** `backend/.env`
2. **מצא את השורה:**
   ```
   WALLET_PRIVATE_KEY=your_private_key_base58_here
   ```
3. **החלף את** `your_private_key_base58_here` **ב-Private Key שהעתקת:**
   ```
   WALLET_PRIVATE_KEY=5KJvsngHeMoi...המפתח המלא שלך...
   ```
4. **שמור את הקובץ** (Ctrl+S)

---

## ✅ שלב 8: בדיקה

1. **הרץ את verify_setup.py:**
   ```bash
   cd backend
   python verify_setup.py
   ```
2. **אמור לראות:**
   ```
   Wallet: [OK] Connected! Address: xxxxx... Balance: X.XXXX SOL
   ```

- ✅ שמרת את ה-Secret Phrase
- ✅ מימנת את הארנק (0.5-1 SOL)
- ✅ ייצאת את ה-Private Key
- ✅ הוספת ל-.env

## ⚠️ אבטחה - כללים חשובים:

1. **לעולם אל תעלה את .env ל-GitHub!** (כבר ב-.gitignore)
2. **שמור את ה-Private Key במקום בטוח** - אם תאבד אותו, תאבד את הגישה לארנק!
3. **השתמש רק בארנק הזה לבוט** - לא לסחר רגיל
4. **התחל עם סכומים קטנים** - רק לבדיקות ($10-20)

## 🆘 בעיות נפוצות:

**Q: לא רואה "Export Private Key"?**
- A: ודא שאתה בארנק הנכון (החדש שיצרת)
- A: חלק מהגרסאות - לחץ על שם הארנק → Advanced → Export Private Key

**Q: Private Key לא עובד?**
- A: ודא שהעתקת את כל המחרוזת (ללא רווחים)
- A: ודא שאין שורות נוספות ב-.env

**Q: איך יודע שהארנק עובד?**
- A: הרץ `python verify_setup.py` - אמור לראות balance
